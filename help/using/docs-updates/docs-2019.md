# Dokumentationsuppdateringar 2019 {#documentation-updates}

En förteckning över ny eller reviderad dokumentation som släppts 2019. Innehåller mindre uppdateringar av Audience Manager-dokumentationen som kanske inte ingår i versionsinformationen för Experience Cloud.

Se [versionsinformationen för Experience Cloud](https://experienceleague.adobe.com/docs/release-notes/experience-cloud/current.html) för ändringar i [!DNL Audience Manager] och andra [!DNL Experience Cloud]-lösningar. Äldre versionsinformation finns i avsnittet [Föregående år](../docs-updates/docs-2018.md).

## November 2019 {#aam-november-updates}

| Hjälpavsnitt | Beskrivning |
|--- |----|
| [Kalifornien CCPA-stöd och översikt över sekretessdokumentation](../overview/data-security-and-privacy/data-privacy.md) | Den [Kalifornien-lagen om konsumentsekretess (CCPA)](https://www.caprivacy.org/about), som träder i kraft den 1 januari 2020, ger personer bosatta i Kalifornien nya rättigheter avseende sina personuppgifter och ålägger dataskyddsansvar för vissa enheter som bedriver verksamhet i Kalifornien. <br> <br> Audience Manager hjälper dig att uppfylla dina skyldigheter enligt sekretesslagstiftningen, via sekretessverktyg som [Adobe Experience Platform Privacy Service](https://docs.adobe.com/content/help/sv-SE/experience-platform/privacy/home.html) för dataåtkomst och borttagningsbegäranden. <br><br> Vi har uppdaterat den aktuella [processen för hantering av avanmälan](../overview/data-security-and-privacy/data-privacy-requests.md#opt-out-requests) så att den inkluderar avanmälan av alla deklarerade ID:n (t.ex. CRM-ID). Om du avanmäler dig via ett deklarerat ID avanmäls det deklarerade ID:t och den senast länkade enheten inte längre för Audience Manager datainsamling. Avanmälningsbegäranden skickar nu även avsegmenteringsbegäranden till [målpartners](../overview/data-security-and-privacy/assets/AAM-Partners-October2019.xlsx) som stöder den här funktionen, både i batch och i realtid. <br><br> Dessutom har vi gjort om vår [Datasäkerhet](../overview/data-security-and-privacy/data-security.md), [Datasekretess](../overview/data-security-and-privacy/data-privacy.md) och [Datastyrning](../overview/data-security-and-privacy/data-governance.md) -dokumentation, så att det blir enklare för dig att hitta den information som krävs för att följa de ovannämnda reglerna. |
